Module luajit-tdlib

A LuaJIT FFI binding for Telegram's Database Library (TDLib)'s JSON interface.

Functions

Functions

initialize(libPath)
Initializes the TDLib library, must be done before creating any client instances.

Parameters:

  • libPath optional string The path into the TDLib shared library, defaults to ./libtdjson.so.

Raises:

Error on library loading failure.

Usage:

    local tdlib = require("luajit-tdlib")
tdlib.initialize("./libtdjson.so")
newClient()
Creates a new instance of TDLib.

Returns:

    Client The created instance of TDLib.

Raises:

Error if tdlib was not initialized.

Usage:

    local tdlib = require("luajit-tdlib")
tdlib.initialize("./libtdjson.so")
local client = tdlib.newClient()
client:destroy()
