Class Client

TDLib client.

Methods

Client:send(request) Sends a request to the TDLib client.
Client:receive(timeout) Receives incoming updates and request responses from the TDLib client.
Client:execute(request) Synchronously executes TDLib request.
Client:isDestoyed() Check if the tdlib client was destroyed or not.
Client:destroy() Destroy the tdlib client, automatically happens in garbage collection.


Methods

Client:send(request)
Sends a request to the TDLib client. May be called from any thread.

Parameters:

  • request table The request content (JSON-serialized internally).

Raises:

Error on json encoding failure, or if the client was destroyed.
Client:receive(timeout)
Receives incoming updates and request responses from the TDLib client. May be called from any thread, but shouldn't be called simultaneously from two different threads.

Parameters:

  • timeout number The maximum number of seconds allowed for this function to wait for new data.

Returns:

    optional table The request response, may be nil if the timeout expires.

Raises:

Error on json decoding failure, or if the client was destroyed.
Client:execute(request)
Synchronously executes TDLib request. May be called from any thread.

Only a few requests can be executed synchronously.

Parameters:

  • request table The request content (JSON-serialized internally).

Returns:

    table The request response.

Raises:

Error on json en/decoding failure, or if the client was destroyed.
Client:isDestoyed()
Check if the tdlib client was destroyed or not.

Returns:

    boolean Whether the client was destroyed or not.
Client:destroy()
Destroy the tdlib client, automatically happens in garbage collection.

Raises:

Error if the client was already destroyed.
