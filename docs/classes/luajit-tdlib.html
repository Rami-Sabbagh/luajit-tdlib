Class luajit-tdlib

A LuaJIT FFI binding for Telegram's Database Library (TDLib)'s JSON interface.

Client:initialize(libPath)
Create a new tdlib client instance. The TDLib library is loaded only once at the first initialize of the first object.

Parameters:

  • libPath optional string The path into the TDLib shared library, defaults to ./libtdjson.so.

Returns:

    luajit-tdlib The created client instance.

Raises:

Error on library loading failure.

Usage:

    local Client = require("luajit-tdlib")
client1 = Client("./libtdjson.so")
client2 = Client()
